# 🌐 Personal Portfolio Website

A modern, responsive portfolio website showcasing my work in **programming, web development, and cybersecurity**.

---

## 🚀 Overview

This portfolio represents my skills, projects, and technical experience.
It is designed with a focus on **performance, clean UI, and smooth user experience** across all devices.

---

## ✨ Features

* 📱 Fully Responsive (Mobile + Desktop)
* ⚡ Fast Loading & Optimized Performance
* 🎯 Clean and Minimal UI/UX
* 📂 Projects Showcase Section
* 📞 Contact Section
* 🌙 Modern Design Style

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Node.js
* **Tools:** Git, GitHub

---

## 📌 Sections

* **Home** – Introduction and quick overview
* **About Me** – Background and skills
* **Projects** – Highlighted work
* **Contact** – Ways to reach me
